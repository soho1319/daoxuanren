# 姓名与五行：你的名字藏着什么能量

名字不只是一个称呼。每天被书写、被呼唤，它就像一道符咒伴随你左右。好的名字如同集福，不好的名字如同诅咒。

## 五行与数字的对应

姓名学中，笔画数对应五行：

| 数字 | 五行 | 特质 |
|------|------|------|
| 一、二 | 木 | 生发调达 |
| 三、四 | 火 | 炎热向上 |
| 五、六 | 土 | 滋养化育 |
| 七、八 | 金 | 清净收敛 |
| 九、零 | 水 | 寒冷向下 |

单数为阳，双数为阴。

## 各五行人的性格特征

### 木（一、二）

**阳木（一）**：大仁大义、耿直心善、上进心强、不屈不挠。善于循序渐进，最终能成为领头人。

**阴木（二）**：心思细腻、善于忍耐、表面温和但内含怒气。有固执的一面，情绪容易内耗。

### 火（三、四）

**阳火（三）**：聪明伶俐、应变力强、性情开朗。重名利、感情来得迅猛，但事情来得快去得也快。

**阴火（四）**：心细、表面温和内心火爆。做事可能没有常性，但一旦认定了就能干出成绩。

### 土（五、六）

**阳土（五）**：温和沉着、有雅量、同情心强、乐善好施。常被称为"老好人"，人缘极好。

**阴土（六）**：稳重、表面温和内心刚毅。外表厚道，能获得相当成功，富有侠义心。但对亲近的人可能缺乏耐性。

### 金（七、八）

**阳金（七）**：讲信用讲义气、重是非。学习能力极强，意志坚定、果敢决断，吃苦耐劳。自我意识强烈。

**阴金（八）**：固执心细、性格刚毅、富有持久力。爱打抱不平，有极强的忍耐力。在好的格局下学业出众。

### 水（九、零）

**阳水（九）**：活动力强、社交广泛、有智慧、富理性。注重权力名誉，掌控欲较强。

**阴水（零）**：喜静、心思沉稳缜密。平时不显山露水，一旦时机来临如山风谷雨，爆发力惊人。

## 五行相生相克

**相生**（顺位滋养）：
火生土 → 土生金 → 金生水 → 水养木 → 木生火

**相克**（隔位制约）：
火克金 → 金克木 → 木克土 → 土克水 → 水克火

名字中各格的五行关系，决定了你人生各方面的顺逆。相生则顺，相克则需要注意对应的方面。

## 五行克制与健康

当名字中出现五行相克的格局时，对应的身体部位容易出问题：

| 克制关系 | 影响部位 |
|----------|----------|
| 金克木 | 肝胆、头、肩 |
| 水克火 | 心脏、牙齿、小肠 |
| 木克土 | 脾胃、鼻、面 |
| 火克金 | 胸、肺、大肠 |
| 土克水 | 肾、膀胱、足 |

这不是说一定会生病，而是这些部位需要多加关注和保养。

## 为什么要用康熙字典

所有字的笔画必须依据**康熙字典中繁体字的笔画**来计算。

因为古人研究姓名理论时还没有简体字，用繁体字才能准确对应五行数理。错一笔、错一划，结果就完全不同。
