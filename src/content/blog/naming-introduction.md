# 姓名学入门：名以定势

## 姓名的意义

姓名学渊源于我国先贤的哲学思想，是中华民族的国粹。

> 行以定名，名以定势，事以验名。

识别事物，必须先有其名，然后才能决定事。事物的发展既成败，又可以验证其名。

老子言：无名，天地之始；有名，万物之母。孔子曰：名正言顺。

**姓名是累生累世积攒因果的体现，当下的有形是背后的无形所决定的。** 姓名传达着天地之玄机。

无论什么人，从摇篮时代到生命的终止，都是名宇深邃——有人因名而得福，有人因名而遭无妄之灾。名字的好坏对一个人的一生都有很大的影响。

## 六格八运法

市面上大多数人用的是三才五格法。我们家不同，用的是**六格八运法**——在三才五格的基础上增加了修格，再加上八运分析，起名时每多加一个条件，不能用的字就多出N倍，所以是**量身定制**。

### 六格

以两个字的名字"张三"为例：

| 格 | 计算 | 说明 |
|---|---|---|
| 天格 | 张+1 | 缺失的要加一去弥补 |
| 人格 | 张三 | — |
| 地格 | 三+1 | — |
| 外格 | 天+地 | 最外边的一相加 |
| **修格** | 张+三 | 人格和地格首尾相加 |
| 总格 | 张+三 | 不加一，所有字相加 |

**修格**——修行的"修"。这是三才五格法没有的格，后天主运和一个人的修行息息相关。

> 万般带不走，唯有业随身。唯有修行可以改命。

修格非常重要，它决定了一个人后天的运势走向。

三才五格法恰恰忽略了修格，所以用三才五格起的名字，修格对应的方面就容易出问题——比如婚姻感情、后运（36岁之后）等。

### 八运

八运是名字在不同年龄段的运势周期分析。通过八运可以看出：

- 每个年龄段需要防范什么
- 交接运时期容易发生什么事
- 哪个阶段是关键期

### 不同字数的列格规则

**三个字的名字**：只需要天格加一去弥补就可以。

**四个字的名字**：四个字为满格，都占满了，不需要加一。

**注意**：修格无论是三个字还是四个字，同样都是人格和地格的首尾相加。总格不加一，所有字相加。

## 重要规则

> 所有字的笔画一定要依据**康熙字典中繁体字的笔画**来列。

因为古人研究姓名理论时还没有我们现在的简体字，所以要用繁体字来记，要更准确一些。错一笔、错一划都不行。

## 量身定制

六格八运法起名，因为条件更多、要求更严，所以出来的名字条件减少很多，很多字都不能用。但也正因为如此，起出来的名字更加精准、安全。

> 名字相当于我们身上的一道符咒，每天都伴随着我们左右。好的名字如同集福，不好的名字如同诅咒。
